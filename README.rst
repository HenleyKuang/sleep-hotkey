Sleep Hotkey
============

Installation
------------

.. code-block:: bash

    pip install sleep-hotkey

How to Use
----------

Call the *sleep* function and hold onto the *ESC* key if you want to stop the sleeping at any time.


Examples
--------

.. code-block:: python

    from sleep_hotkey.sleep_hotkey import sleep
    sleep(10)
